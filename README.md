# IDE建议

如果你会java或python就最好使用java和python

- 因为：
    - c和c++需要我们自己进行内存的申请和释放，会让我们花一些时间去处理这里没有意义的事情
    - 而java和python有垃圾回收机制也就是GC，让我们可以把注意力更多的花在程序上
